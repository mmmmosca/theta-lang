# Theta Examples

Run any example with:

```powershell
python .\theta.py .\examples\<file>.th
```

Included:
- `double_list.th` — pattern matching with `matches` and star-rest
- `factorial.th` — recursion with `when/else`
- `fibonacci.th` — recursion with `when/else`
- `pattern_sum2.th` — list pattern examples
- `tm_increment.th` — Turing machine blueprint demo
- `python_blueprint.th` — calling into Python via blueprint
- `io_examples.th` — arrays, indexing, printing
